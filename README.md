## About

Installation

```bash
sudo snap install hugo --channel=extended/stable
```

Running server

```bash
hugo server -D
```

Creating new post:

```bash
hugo new posts/my-first-post.md
```

Updating theme:

We are using https://github.com/theNewDynamic/gohugo-theme-ananke

```bash
git submodule update --remote --merge
```
